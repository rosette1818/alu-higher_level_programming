# python-network_1

Project: Python - Network #1

## Description

This project covers fetching internet resources with Python, using
both the standard library `urllib` package and the third-party
`requests` package. It includes making GET and POST requests,
reading and decoding response bodies, reading response headers,
handling HTTP errors, working with JSON APIs, and using Basic
Authentication against the GitHub API.

## Requirements

- Editors: vi, vim, emacs
- Interpreted on Ubuntu using python3
- All files end with a new line
- First line of all files: `#!/usr/bin/python3`
- Code follows the PEP 8 style guide
- All files are executable
- All modules have documentation
